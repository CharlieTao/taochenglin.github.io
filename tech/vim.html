# vim

gg：光标跳转到该文件的行首； dG：删除光标行及其以下行的全部内容。

yy是复制光标所在行，nyy是复制包括光标所在行以及向下的n-1行，一共n行。 p是将已经复制的数据，粘贴在光标所在行的下一行。大写P为粘贴在光标所在行的上一行。

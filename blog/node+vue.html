    # vue

    在用脚手架搭建vue项目时，提示：无法加载文件 C:\Users\1111111\AppData\Roaming\npm\vue.ps1,因为在此系统禁止运行脚本

    因为禁止运行脚本，解决办法

    1.管理员身份打开PowerShell

    2.输入set-ExecutionPolicy RemoteSigned

    3.选择Y 或者A
